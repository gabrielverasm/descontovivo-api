# ADR-0002: Usar Java 25 LTS

## Status

Aceita

## Contexto

O projeto será criado do zero e deve servir como vitrine profissional junto com o frontend `descontovivo-ui`.

Java 21 LTS foi considerado pela estabilidade e ampla adoção corporativa.

## Decisão

Usaremos Java 25 LTS.

## Consequências

- O projeto nasce usando o LTS mais recente do Java.
- A stack fica mais atual para portfólio.
- Será necessário garantir compatibilidade com Quarkus 3.33.x LTS e dependências escolhidas.
