# ADR-0001: Usar Quarkus LTS

## Status

Aceita

## Contexto

O DescontoVivo API precisa ser moderno, performático, produtivo e adequado para uma vitrine profissional de backend Java.

## Decisão

Usaremos Quarkus 3.33.x LTS como framework principal.

## Consequências

- Melhor alinhamento com aplicações cloud native.
- Boa integração com PostgreSQL, Panache, OpenAPI, Health, OIDC e testes.
- Menor boilerplate em comparação com stacks tradicionais.
